Doe rechterklik in in de map waar je deze readme vond. 
"klik meer opties weergeven"
Klik op "Open with Code"
Houd control ingedrukt en druk op ` (hint: deze knop zit onder escape)
Als het goed is komt er een terminal omhoog onderaan het scherm (zoniet doe dan nog eens ctrl + `) 
typ in het terminal "node index" en wacht tot het script klaar is.


Alle afbeeldingen uit de map "afbeeldingen-origineel" zijn hebben nu de achtergrond verwijderd gekregen en gekopieerd naar de afbeeldingen-verwerkt map met een witte achgtergrond. 

LET OP: deze actie kost credits per afbeelding... Verwijder daarom ook meteen alle afbeeldingen uit afbeeldingen-origineel'!
